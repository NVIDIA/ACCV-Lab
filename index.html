ACCV-Lab Documentation

Welcome to ACCV-Lab!

Accelerated Computer Vision Lab (ACCV-Lab) is a systematic collection of packages with the common goal to facilitate end-to-end efficient training in the ADAS domain, each package offering tools & best practices for a specific aspect/task in this domain.

Please see ACCV-Lab Overview for an overview of the project, including a short overview of the contained packages and a Quick Start section, as well as the more detailed guides on how to use this project (Installation Guide, Development Guide, …) and further information.
